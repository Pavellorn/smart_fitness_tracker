# 🏃‍♂️ Умный Фитнес Трекер (Kivy Mobile App)

[![Kivy](https://img.shields.io/badge/Kivy-2.3+-brightgreen)](https://kivy.org)
[![Python](https://img.shields.io/badge/Python-3.8+-blue)](https://python.org)
[![License](https://img.shields.io/badge/License-MIT-yellow)](LICENSE)

Мобильное приложение для отслеживания тренировок с интуитивным интерфейсом, плавными анимациями и сохранением данных. Проект разработан с нуля по техническому заданию заказчика.

## 📱 Основные возможности

### ✅ Реализованный функционал

| Экран | Функции |
|-------|---------|
| **Тренировка** | • Анимированный круг прогресса (0→100%)<br>• Пульсирующая иконка сердца<br>• Таймер на 30/60/90 минут<br>• Автосохранение прогресса |
| **Статистика** | • График активности за 7 дней<br>• Итоговое время за неделю<br>• Определение лучшего дня |
| **Настройки** | • Цель тренировок (слайдер 100-500 мин/нед)<br>• Toggle уведомлений<br>• Светлая/тёмная тема<br>• Сброс данных |

### ✨ Особенности
- **Плавные анимации**  пульсация сердца, градиентная заливка круга
- **Свайп-навигация** между экранами
- **JSON-сохранение** всех данных (работает после перезапуска)
- **Адаптивный дизайн** (относительные размеры)

## 🛠️ Технологии

- **Python 3.8+**
- **Kivy 2.3+** (KV язык разметки)
- **Kivy Storage** (JsonStore)

## 🚀 Установка и запуск

### Предварительные требования
- Python 3.8 или выше
- pip (менеджер пакетов)

### Шаги по установке

1. **Клонируйте репозиторий**
```bash
git clone https://github.com/Pavellorn/smart_fitness_tracker
cd fitness-tracker
```

2. **Установите Kivy и зависимости**
```bash
pip install kivy==2.3.0
```

3. **Запустите приложение**
```bash
python main.py
```

## 🎮 Как пользоваться

### Экран тренировки
1. Выберите длительность (30/60/90 минут)
2. Нажмите "Старт" - начнётся обратный отсчёт
3. Круг заполняется по мере выполнения, сердечко пульсирует

### Экран статистики
- График показывает активность за последние 7 дней
- Внизу отображается общее время
- Данные обновляются автоматически после тренировок

### Экран настроек
- Установите недельную цель (слайдер)
- Включите/отключите уведомления
- Переключите тему (светлая/тёмная)
- Кнопка "Сбросить" очищает статистику за неделю
